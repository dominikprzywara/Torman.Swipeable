# Torman.Swipeable
Xamarin.Android custom controls with swipe actions.

![alt text][swipeAnimation]

## Download
[NuGet version](https://www.nuget.org/packages/Torman.Swipeable).

## Demo app
You can find sample app in sample/ directory.

* Different animations.
